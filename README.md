# Setup Script
This script will automatically setup a python project.

## Features
- Creates git repo
- Creates README and LICENSE
- Creates Virtual Environment
- Installs pytest, pytest-cov, flake8
- Creates src and test directory
- Creates inital commit

## Installation
```bash
git clone https://www.github.com/TheWhiteLionGod/SetupScript
mv SetupScript/setupscript.sh {PROJECT}
cd {PROJECT} && chmod +x setupscript.sh
```

## Usage
```bash
./setupscript.sh
```
